## ✨ Demonstração ao vivo

A interface estática do núcleo está disponível em:

> ⚠️ **Nota**: A demonstração exibe a interface, mas as interações com o backend (chat, pensamentos dinâmicos) funcionam apenas localmente. Para a experiência completa, execute o projeto no seu computador seguindo as instruções abaixo.

## 🚀 Funcionalidades

| Módulo | Descrição |
|--------|-----------|
| 😊 **Emoção contínua** | Humor emergente (confuso, curioso, reflexivo, analítico) baseado em conflito e novidade |
| 🧠 **Memória episódica** | Curto e longo prazo, com consolidação automática |
| ⚙️ **Drives internos** | Vontades de novidade, coerência e competência |
| 🎯 **Meta‑objetivo adaptativo** | O núcleo redefine sua própria meta conforme o histórico |
| 🌐 **API REST** | Endpoints `/think`, `/ask`, `/export` (CSV/JSON) |
| 🖥️ **Dashboard 3D** | Avatar Three.js, gráfico radar, barras de drives, chat |
| 🧪 **Testes automatizados** | `node test-all.js` valida tudo |
